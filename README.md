# Financial Planning Calculator

A simple and interactive web-based tool for financial planning, featuring calculators for both Systematic Investment Plans (SIP) and Systematic Withdrawal Plans (SWP). This project helps users visualize and understand the potential growth of their investments and the sustainability of their withdrawals.

## Features

- **SIP Calculator**: Calculate the future value of your monthly investments based on expected returns and duration.
- **SWP Calculator**: Determine the sustainability of regular withdrawals from a corpus, showing the remaining balance over time.
- **Interactive Visualizations**: Each calculator includes a dynamic chart that visualizes the growth of your investments (SIP) or the depletion of your corpus (SWP) over time.
- **Responsive Design**: The interface is built to be user-friendly on both desktop and mobile devices.
- **Simple and Intuitive Interface**: The design focuses on ease of use, with clear input fields and instant results.

## Technologies Used

- **HTML5**: For the basic structure of the web page.
- **CSS3**: For styling the components.
- **Tailwind CSS**: A utility-first CSS framework used for rapid UI development and responsive design.
- **JavaScript (ES6)**: To handle all the calculation logic and user interactions.
- **Chart.js**: A powerful charting library used to create the data visualizations.

## How to Use

### Clone the Repository

```bash
git clone https://github.com/sahil-shahare/Financial-Planning-Calculator.git
```

### Open the File

Simply open the `index.html` file in your preferred web browser. No server is required.

## Using the Calculators

### SIP Calculator
1. Enter your desired **Monthly Investment**.
2. Input the **Expected Annual Returns** (in percentage).
3. Specify the **Investment Duration** (in years).
4. Click **"Calculate SIP"** to see your total investment, the final value, and a growth chart.

### SWP Calculator
1. Enter your **Total Corpus**.
2. Input the **Monthly Withdrawal amount**.
3. Specify the **Expected Annual Returns** (in percentage) on the corpus.
4. Enter the **Withdrawal Duration** (in years).
5. Click **"Calculate SWP"** to view the total amount you will withdraw and the remaining corpus, along with a visualization of the corpus over time.
